Automated Classroom Management System



service :

Web Application

started :

8 Aug 2022

completed :

13 Dec 2022

Automated Classroom Management System is a web application project that has been done as a part of our course work for software project lab 1.
In this project, we have tried to build a website that can fetch data from google classroom, Microsoft Teams. Through this website students are able to find necessary materials, assignment deadlines and other required notices all in one place without getting into other classroom management websites used by faculty members. We have integrated Google Classroom API to fetch necessary data for this website.

Since in SPL1 we are not allowed to use any framework so the Tech stacks used are HTML, CSS, JS, MySQL as database, and php in the backend
