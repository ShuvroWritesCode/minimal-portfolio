Jersey Merchant e-commerce is a web application for Jersey Merchant that has been done as a part of our course work for software project lab 2. In this project, we are building Jersey Merchant e-commerce website.
Jersey Merchant is a social media based business that is selling sportswear and sports accessories for over 3 years now. With over 100 customer queries per day, an e-commerce web solution was necessary for the business. As our project, we took the challenge to build a modern e-commerce website with complete functionality for Jersey Merchant.

Tech stacks used are NextJS, NodeJS, Postgres and Tailwind CSS
